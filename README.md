# Network Fitness Survey

An interactive web-based survey for assessing fiber network management health and identifying improvement opportunities.

## Features

- Visual rating system (1-5 scale)
- Automated scoring and recommendations
- Responsive design for all devices
- No dependencies - pure HTML/CSS/JS

## Usage

Simply open `index.html` in any modern browser, or visit the [GitHub Pages site](https://rossperry22-jpg.github.io/network-fitness-survey/).

## Technology

- HTML5
- CSS3 (with CSS Grid & Flexbox)
- Vanilla JavaScript
- No frameworks or external dependencies
